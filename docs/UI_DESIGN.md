# Polaris 安静单色设计系统（V2.0）

> 视觉语言 4.0：单色（北极星靛蓝）+ 安静动效。移除全部商业平台视觉特征（奖励色、count-up、粒子、scaleIn、学段自适应 token），回归"安静、可信、可随身携带的自学伙伴"。所有设计令牌定义在 `src/index.css`。

## 1. 设计原则

V2.0 视觉语言 4.0 遵循四条原则：

- **安静 > 喧闹**：移除所有粒子、count-up、scaleIn 强调动画，保留页面转场、卡片 hover 微抬升、按钮按压反馈。
- **单色 > 多彩**：主色 `#6366F1`（北极星靛蓝），辅助 `#0B0F19`（深空底）、`#F8FAFC`（亮色文字）、`#94A3B8`（次要文字）。删除"星光琥珀 #F59E0B"奖励色。
- **留白 > 密集**：卡片间距从 16px 提到 24px，行高从 1.5 提到 1.7，正文最大宽度限制 72 字符。
- **本地字体 > 装饰字体**：保留 Inter Variable（正文）+ Sora（标题），删除按学段动态调整字重逻辑。

## 2. 颜色令牌

### 2.1 核心调色板（单色系统）

| Token | 值 | 用途 |
|-------|-----|------|
| `--polaris-bg` | `#0B0F19` | 页面主背景（深空） |
| `--polaris-surface` | `#11131C` | 卡片、面板表面 |
| `--polaris-surface-elevated` | `#171A25` | 抬升表面、Popover、Sheet |
| `--polaris-accent` | `#6366F1` | 主按钮、选中状态、链接、强调（北极星靛蓝，唯一强调色） |
| `--polaris-accent-glow` | `#818CF8` | 悬停光晕、脉冲 |
| `--polaris-text-primary` | `#F8FAFC` | 主要文本（亮色） |
| `--polaris-text-secondary` | `#94A3B8` | 次要文本、说明 |
| `--polaris-text-muted` | `#64748B` | 禁用、占位符 |

### 2.2 语义颜色

| 语义 | 值 | 用途 |
|------|-----|------|
| Success | `#10B981` | 答对、掌握（仅状态指示，不作为装饰） |
| Danger | `#EF4444` | 答错、薄弱节点、删除 |
| Warning | `#F59E0B` | 仅保留用于系统级警示（不再是"奖励色"） |

> V2 删除"星光琥珀 #F59E0B"作为奖励色出现的位置（XP、星光、徽章、连胜、宝箱均已随商业特征移除）。Warning 语义仅用于真正的警示场景，不用于装饰。

### 2.3 玻璃拟态令牌

| Token | 值 | 用途 |
|-------|-----|------|
| `--glass-bg` | `rgba(17, 19, 28, 0.72)` | 玻璃背景（仅浮层） |
| `--glass-border` | `rgba(255, 255, 255, 0.08)` | 玻璃边框高光 |

玻璃拟态仅用于浮层（Sheet、Popover、Dialog），主表面使用实色卡片以保证对比度。

## 3. 字体与排版

### 3.1 字体家族

| Token | 值 | 用途 |
|-------|-----|------|
| `--font-inter` | `'Inter Variable', 'Inter', "PingFang SC", "Microsoft YaHei", ...` | 正文、界面文本 |
| `--font-display` | `'Sora', sans-serif` | 标题、数字、品牌展示 |
| `--font-mono` | `"SF Mono", "Cascadia Code", ...` | 版本号、代码、日志 |

所有字体本地托管在 `public/fonts/`，构建后进入 `dist/fonts/`，零外部网络依赖。V2 删除按学段动态调整字重的逻辑，全应用统一一套字重阶梯。

### 3.2 排版规范

| 规范 | 值 | 说明 |
|------|-----|------|
| 正文字号 | 16px | 默认 `--text-base` |
| 行高 | 1.7 | V2 从 1.5 提到 1.7，降低阅读疲劳 |
| 正文最大宽度 | 72 字符 | 长文阅读区限宽，避免视线跨度过大 |
| 卡片间距 | 24px | V2 从 16px 提到 24px，增强留白 |
| 段落间距 | 1em | 段落间空一行 |

### 3.3 字号阶梯

| Token | 默认 | 用途 |
|-------|------|------|
| `--text-xs` | 12px | 辅助说明、时间戳 |
| `--text-sm` | 14px | 次要文本、表格 |
| `--text-base` | 16px | 正文 |
| `--text-lg` | 18px | 小标题 |
| `--text-xl` | 20px | 区块标题 |
| `--text-2xl` | 24px | 页面标题 |
| `--text-3xl` | 30px | Sora 展示标题 |

## 4. 组件层级

### 4.1 卡片

- 表面：`--polaris-surface`（实色，非玻璃）
- 圆角：`12px`（统一，无学段差异）
- 内边距：`20px`（紧凑信息）/ `24px`（主内容）
- 间距：卡片间 `24px`
- 边框：`1px solid rgba(255,255,255,0.06)`（极淡高光）
- hover：仅 `translateY(-2px)` + 阴影加深，无 scale spring

### 4.2 按钮

| 层级 | 样式 | 用途 |
|------|------|------|
| Primary | `--polaris-accent` 实色填充 + 白字 | 主操作（继续学习、复习错题） |
| Secondary | `--polaris-surface-elevated` 填充 + 主文字 | 次操作 |
| Ghost | 透明 + accent 文字 | 链接、取消 |
| Danger | Danger 色描边 + Danger 文字 | 删除 |

按压反馈：`scale(0.98)` + 150ms ease-out。无 ripple、无 glow 脉冲。

### 4.3 知识地图节点（3 态）

| 状态 | 视觉 |
|------|------|
| 已掌握 | `--polaris-accent` 实色填充 + 白字 |
| 学习中 | 透明 + `--polaris-accent` 描边 + 主文字 |
| 薄弱 | 透明 + `--polaris-accent` 虚线描边 + 浅色填充 |

> 删除 v1.0 的"未解锁"状态、"星云遮蔽"、"裂纹衰减"、"红光裂纹"等游戏化视觉。

## 5. 学段配置（无视觉差异）

V2 学段从 5 档简化为 3 档，**仅影响 AI 老师的 prompt 风格**（用词深度、举例偏好），不驱动任何视觉差异：

| 学段 | 标识 | 影响 |
|------|------|------|
| YOUTH | 少年 | AI prompt 偏向具象举例、浅显用词 |
| TEEN | 青少年 | AI prompt 平衡抽象与具象 |
| ADULT | 成年 | AI prompt 偏向抽象、专业术语 |

`learning-modes.ts` 仅保留 prompt 风格配置，删除 `data-mode` 切换的圆角 / 字号 / 插画密度 token。`migrateLearningMode` 兼容旧 5 档（KINDERGARTEN/ELEMENTARY/MIDDLE → YOUTH，HIGH → TEEN，PROFESSIONAL → ADULT）。

## 6. 页面布局

### 6.1 安静桌面首页

```
┌─────────────────────────────────────┐
│  问候语 + 日期            今日学习时长 │  ← 顶部，小字
│                                       │
│  ┌───────┐ ┌───────┐ ┌───────┐      │
│  │继续学习│ │问 AI  │ │复习错题│      │  ← 3 入口，简洁卡片
│  └───────┘ └───────┘ └───────┘      │
│                                       │
│           小灵形象 + 鼓励语            │  ← 底部
└─────────────────────────────────────┘
```

删除：Bento Grid、今日任务卡、连胜统计、XP 进度条、双货币余额、知识星图进度环、推荐练习、错题本快捷入口。

### 6.2 AI 老师对话

消息气泡 + Markdown/LaTeX 渲染 + 流式逐字 + 工具栏（朗读 / 复制 / 重新生成 / 清空）。无环形阶段进度、无 stage 标签解析。

### 6.3 内容最大宽度

正文区与对话区均限制最大宽度，保证阅读舒适：

- 首页入口区：`max-width: 720px`
- AI 对话气泡：`max-width: 72 字符`
- 学习数据图表区：`max-width: 960px`

## 7. 禁用清单（V2 移除）

以下视觉元素在 V2 中**禁止出现**：

- 星光琥珀 `#F59E0B` 作为奖励 / 强调色
- count-up 数字滚动动画
- 粒子扩散、星光粒子
- 徽章 scaleIn 入场
- 连胜 pulse-glow 光晕
- 阶段高亮脉冲（苏格拉底 6 阶段）
- 错题粒子反馈
- 宝箱开启 Lottie
- `ease-out-expo` 缓动（改用 `ease-out`）
- 学段自适应圆角 / 字号 / 插画密度 token

> 动效规范详见 [ANIMATION.md](./ANIMATION.md)。
