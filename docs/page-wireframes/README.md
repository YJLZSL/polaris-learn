## 1. 首页

> 对应代码：`lib/features/home/home_page.dart`
> 角色：登录后落地页，承载「今日继续学习」入口、Streak 打卡、吉祥物陪伴首屏。

### 1.1 信息层级（从上到下）

```
┌─────────────────────────────────────────┐
│ LingxiAppBar                            │ ← 顶部导航（含 Logo + 主题切换）
├─────────────────────────────────────────┤
│                                         │
│            ┌─────────┐                  │
│            │ 吉祥物   │  200px           │ ← 情感锚点（idle 状态呼吸动画）
│            │  小犀    │                  │
│            └─────────┘                  │
│                                         │
│         "下午好，继续探索吧"              │ ← headlineMedium
│                                         │
│      ┌──────────────────────┐           │
│      │ 🔥 7 天连续打卡       │           │ ← Streak 徽章（streakFire #FF5722）
│      └──────────────────────┘           │
│                                         │
│      ┌──────────────────────┐           │
│      │ ▶ 继续上节课          │           │ ← 主 CTA（FilledButton）
│      │   L1 · 第 3 节        │           │
│      └──────────────────────┘           │
│                                         │
│      快捷入口（2×2 网格）                │
│   ┌────────┐  ┌────────┐                │
│   │ 📚 学习 │  │ 💬 对话 │              │
│   └────────┘  └────────┘                │
│   ┌────────┐  ┌────────┐                │
│   │ 📝 笔记 │  │ 🏆 成就 │              │
│   └────────┘  └────────┘                │
│                                         │
├─────────────────────────────────────────┤
│ NavigationBar（首页/学习/对话/笔记/我的）│ ← 桌面端收为左侧 NavigationRail
└─────────────────────────────────────────┘
```

### 1.2 组件选型

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶栏 | `LingxiAppBar` | 统一应用栏，含 Logo + 主题切换图标 |
| 吉祥物 | `MascotWidget(size: 200)` | 点击 5 次触发 celebrate 彩蛋 |
| 欢迎语 | `Text` + `headlineMedium` | 根据时段切换问候（早/午/晚） |
| Streak 徽章 | 自定义 `StreakBadge` | 火焰图标 + 天数，`streakFire` 色 |
| 主 CTA | `FilledButton.icon` | 有未完成课时才显示，否则隐藏 |
| 快捷入口 | `LingxiCard` × 4 | 圆角 16，点击跳转对应路由 |
| 底部导航 | `NavigationBar` / `NavigationRail` | 5 项，当前项高亮 |

### 1.3 交互流

| 触发 | 行为 | 目标路由 |
|------|------|----------|
| 点击吉祥物 | 切换 mood（idle→happy→curious） | 无（本地状态） |
| 连点吉祥物 5 次 | 触发 celebrate 动画 + 触觉反馈 | 无（彩蛋） |
| 点击「继续上节课」 | 携带 lessonId 跳转 | `/learning/lesson/{id}` |
| 点击「学习」快捷卡 | 跳转学习路径 | `/learning` |
| 点击「对话」快捷卡 | 跳转对话列表 | `/chat` |
| 点击「笔记」快捷卡 | 跳转笔记列表 | `/notes` |
| 点击「成就」快捷卡 | 跳转成就页 | `/progress/achievements` |

### 1.4 桌面 / 移动布局差异

**移动端（< 600）：**
- 单列垂直布局，内容居中，maxWidth 600
- 底部 `NavigationBar`（5 项）
- 吉祥物 200px，快捷入口 2×2 网格

**平板端（600–1023）：**
- 内容居中，maxWidth 720
- 底部 `NavigationBar`
- 吉祥物 220px，快捷入口 2×2 网格（卡片更宽）

**桌面端（≥ 1024）：**
- 左侧 `NavigationRail`（收起态 72px / 展开态 240px）
- 主内容区居中，maxWidth 960
- 吉祥物 240px，快捷入口 4×1 横排
- 右侧可放「今日学习摘要」侧栏（320px）

> 角色：展示 L0–L4 五级课程树，是学习主入口。

```
│                                         │

│  │ │                                │    │
│  │ ○  L2  深入者                    │    │ ← 未解锁（灰色空心圆）
│  │ │  🔒 需完成 L1                   │    │
│  │ │                                │    │

│  │    🔒 需完成 L3                   │    │
│  └─────────────────────────────────┘    │
│                                         │
│  竖线连接各节点（timeline 样式）          │
│                                         │
└─────────────────────────────────────────┘
```

### 2.2 组件选型

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶栏 | `LingxiAppBar(title: Text('学习路径'))` | — |
| 进度摘要 | `LinearProgressIndicator` + `Text` | 全局完成百分比 |
| 课程节点 | 自定义 `LearningLevelNode` | 圆点 + 竖线 timeline |
| 状态标记 | `Icon`（locked/unlocked/in-progress） | 三态视觉差异 |
| 继续 CTA | `FilledButton.tonal` | 仅进行中级显示 |

### 2.3 交互流

| 触发 | 行为 | 目标路由 |
|------|------|----------|
| 点击已解锁级别 | 展开该级别课程列表 | 本页内展开 |
| 点击「继续」 | 跳转到当前级别下一未完成课时 | `/learning/lesson/{lessonId}` |
| 点击锁定级别 | SnackBar 提示「需先完成 L{n-1}」 | 无 |
| 长按级别节点 | 弹出该级别详情对话框 | 无 |

### 2.4 桌面 / 移动布局差异

**移动端：**
- 单列垂直 `ListView`，节点占满宽度
- 竖线 timeline 在左侧 24px 处
- 节点卡片 padding 16

**平板端：**
- 单列，maxWidth 720 居中
- 节点卡片 padding 20

**桌面端：**
- 双列：左侧 timeline（主路径 480px）+ 右侧「级别详情」面板（自适应）
- 点击左侧节点时右侧面板展示该级别所有课时缩略卡片
- 竖线 timeline 在左侧 32px 处，节点更大

### 3.2 组件选型

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶栏 | `LingxiAppBar` + 返回按钮 | 含「下一节」菜单 |

| 知识点卡片 | `LearningCardWidget` | 主图渐变 + 核心解释 + 「为什么重要」 |
| 分级按钮 | `Chip` / `ActionChip` × 3 | L0 简化 / L2 深入 / L3 图解 |
| 测验 | `QuizWidget` | 单选/多选/填空，阈值 `kQuizPassThreshold=0.8` |
| 苏格拉底面板 | `SocraticDialogPanel` | 桌面右侧 / 移动底部 |
| 继续学习侧栏 | `ContinueLearningSidebar` | 仅桌面端，280px |

| 触发 | 行为 | 目标 |
|------|------|------|
| 左右滑动主内容 | 切换知识点 | 本页 PageView |
| 点击分级按钮 | 切换解释深度（同卡片内刷新） | 本页状态 |
| 提交测验答案 | 即时判分，≥80% 显示庆祝 | 触发 mascot.celebrate |
| 测验未通过 | 显示错误解析 + 允许重试 | 本页状态 |
| 点击「完成本节」 | 标记完成 + 解锁下一节 + 检查成就 | 返回学习路径或进入下一节 |
| 苏格拉底对话发送 | AI 流式回复 + mascot 联动 | 本页消息列表 |
| 桌面侧栏点击「下一节」 | 跳转 | `/learning/lesson/{nextId}` |

### 3.4 桌面 / 移动布局差异

**移动端：**
- 单列 `PageView`，知识点全屏滑动
- 苏格拉底面板在知识点下方（可折叠）
- 继续学习侧栏隐藏，改为右下角 FAB 打开底部抽屉
- 测验紧跟最后一个知识点

**平板端：**
- 单列，maxWidth 720
- 苏格拉底面板仍折叠在底部

**桌面端：**
- 三列布局：主内容（760）+ 继续学习侧栏（280）+ 苏格拉底面板（360）
- 知识点 `PageView` 在主内容区
- 苏格拉底面板常驻可见
- 继续学习侧栏常驻可见

## 4. 对话页

> 对应代码：`lib/features/chat/chat_page.dart`、`chat_controller.dart`
> 角色：与 AI 一对一对话，流式输出，吉祥物联动。

### 4.1 信息层级（从上到下）

```
┌─────────────────────────────────────────┐
│ ← 返回   对话标题              小犀(80)  │ ← 顶栏（吉祥物缩略 + 标题）
├─────────────────────────────────────────┤
│                                         │
│            ┌──────────────┐             │
│            │ AI 气泡       │             │ ← AI 消息（左对齐，圆角不对称）
│            │ "你好..."    │             │
│            └──────────────┘             │
│                                         │
│                      ┌──────────────┐   │
│                      │ 用户气泡      │   │ ← 用户消息（右对齐，主色背景）
│                      │ "帮我解释..." │   │
│                      └──────────────┘   │
│                                         │
│            ┌──────────────┐             │
│            │ AI 正在输入▊  │             │ ← 流式光标（500ms 闪烁）
│            └──────────────┘             │
│                                         │
│         （反向 ListView，新消息在底部）   │
│                                         │
├─────────────────────────────────────────┤
│ ┌─────────────────────────────┐ [发送]  │ ← 输入区（固定底部）
│ │ 输入消息...                  │         │
│ └─────────────────────────────┘         │
└─────────────────────────────────────────┘
```

### 4.2 组件选型

| 区域 | 组件 | 说明 |
|------|------|------|
| 顶栏 | `LingxiAppBar` | 含返回 + 标题 + 吉祥物缩略（80px） |
| 消息列表 | 反向 `ListView` | 新消息在底部，自动滚动 |
| AI 气泡 | `Container` + 不对称圆角 | `borderRadius: 16 16 16 4` |
| 用户气泡 | `Container` + 主色背景 | `borderRadius: 16 16 4 16` |
| 流式光标 | `AnimatedOpacity` | 500ms 闪烁 |
| 输入框 | `TextField` + `FilledButton.icon` | 发送按钮 |
| 吉祥物悬浮 | `MascotOverlay` | AI 思考时右下角 80px |

### 4.3 交互流

| 触发 | 行为 | 目标 |
|------|------|------|
| 发送消息 | 清空输入 + AI 流式回复 + mascot.setAiThinking | 本页消息列表 |
| AI 开始回复 | mascot 切换 thinking + 右下角悬浮 | 本页状态 |
| AI 回复完成 | mascot 切换 happy | 本页状态 |
| 长按消息 | 弹出菜单（复制/保存为笔记/删除） | 本页操作 |
| 点击顶栏吉祥物 | 跳转到 mascot 互动 | 本页彩蛋 |
| 点击顶栏标题 | 编辑对话标题 | 本页对话框 |

### 4.4 桌面 / 移动布局差异

**移动端：**
- 全屏单列，消息气泡 maxwidth 75%
- 输入区固定底部

**平板端：**
- 单列，maxWidth 720 居中
- 气泡 maxwidth 70%

**桌面端：**
- 居中单列，maxWidth 800
- 左侧可展开「历史对话列表」抽屉（280px）
- 气泡 maxwidth 65%
- 输入区更宽，支持多行

| 区域 | 组件 | 说明 |
|------|------|------|
| 容器 | `Center` + `Column` | 垂直居中 |
| 吉祥物 | `MascotWidget(size: 160, enableTapInteraction: false)` | mood 由场景决定 |
| 标题 | `Text` + `headlineSmall` | 简短描述「还没有 X」 |
| 描述 | `Text` + `bodyMedium` | 引导文案 |
| CTA | `FilledButton` | 主行动按钮 |
| 通用组件 | `EmptyStateWidget` | 统一封装，参数化 mood/title/desc/cta |

### 11.3 交互流

| 触发 | 行为 | 目标路由 |
|------|------|----------|
| 点击 CTA | 跳转到对应入口 | 由 `onCta` 回调决定 |
| 点击吉祥物 | 触发 mood 切换 | 本页彩蛋 |

## 附录：线框图使用约定

### A1. ASCII 字符约定

| 字符 | 含义 |
|------|------|
| `┌ ┐ └ ┘` | 卡片/容器边框（四角） |
| `─ │` | 边框线（横/竖） |
| `├ ┤ ┬ ┴ ┼` | 边框交叉点 |
| `▓ ░` | 填充/进度（深/浅） |
| `● ○` | 已完成/未完成节点 |
| `◀ ▶` | 左右切换 |
| `▼ ▲` | 展开/折叠 |
| `...` | 内容省略 |

### A2. 尺寸标注约定

- 所有尺寸均为逻辑像素（dp），非物理像素
- 移动端默认参考宽度 360（Android 小屏）
- 桌面端默认参考宽度 1280
- 断点严格遵循 `Responsive.isMobile/isTablet/isDesktop`（600/1024）

### A3. 与 design-tokens.json 的对应

- 颜色：引用 `color.*` 令牌
- 字体：引用 `typography.*` 令牌
- 间距：引用 `spacing.*` 令牌（sm=8, md=16, lg=24, xl=32）
- 圆角：引用 `radius.*` 令牌（sm=8, md=12, lg=16, xl=24, full=9999）
- 动效：引用 `motion.*` 令牌（springDefault/Fast/Slow）

### A4. 重设计优先级建议

| 优先级 | 页面 | 理由 |
|--------|------|------|
| P0 | 首页、课程页、对话页 | 用户最高频接触 |
| P1 | 学习路径页、引导页 | 决定首次印象 |
| P2 | 笔记页、设置页、成就页 | 功能性页面 |
| P3 | 统计页、帮助页、空状态 | 低频但影响完整度 |
